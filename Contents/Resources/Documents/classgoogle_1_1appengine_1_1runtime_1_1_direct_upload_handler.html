Detailed Description

Handle direct file uploads by placing contents in virtual file system.

The PHP runtime has been modified (in rfc1867.c) to place the file contents in the 'contents' key of each $_FILES entry. The handle() method moves the contents into a virtual file system accessed via stream wrapper (vfsStream) and alters the tmp_name to point to vfs://.
