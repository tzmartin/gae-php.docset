google\appengine\ext\cloud_storage_streams\ini_get (   $varname)

Gets the value of a configuration option.

Override built-in ini_get() to fake INI value that would normally be provided by gae extension, but is not on devappserver. INI will always be true during these tests.

  • google_app_engine.enable_additional_cloud_storage_headers: true
Parameters
string$varnameThe configuration option name.
Returns
mixed Returns the value of the configuration option as a string on success, or an empty string for null values. Returns FALSE if the configuration option doesn't exist.
